## Description

Tim Hortons Run is a SwiftUI application that allows team members to create, save, and manage coffee orders for a daily Tim Hortons run.

## Features

* Create and save coffee orders
* Select drink size and type
* Customize sugar and cream
* Save favourite orders
* Delete saved orders
* Coffee Run Timer
* Cart / Order Confirmation screen
* Tim Hortons themed splash screen
* Custom Tim Hortons logo
* JSON persistence using UserDefaults
